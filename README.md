#### Metasyntactic variables

Used as meaningless placeholder. 
Wikipedia [page](https://en.wikipedia.org/wiki/Metasyntactic_variable) about them.
Several english variables of that kind:

`['foo', 'bar', 'baz', 'qux', 'quux', 'corge', 'grault', 'garply', 'waldo', 'fred', 'plugh', 'xyzzy', 'thud']`

Vim script, which define some hot keys for past it [there](https://gist.github.com/hail2u/6561431).

#### Text samples

When will need some text samples for testing working with text, 
[there](https://github.com/cs50/problems/archive/speller.zip) a few cheap text files. In `problems-speller\texts`
a lot of various text files, which vary by size and content. There is `problems-speller\dictionaries\large` text file,
which present in one word per line format. All words in `problems-speller\dictionaries` are *unique*.

For downloading:

`wget https://github.com/cs50/problems/archive/speller.zip`
